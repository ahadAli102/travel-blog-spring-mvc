# travel-blog-spring-mvc
## Travel blog app make using JAVA with SPRING MVC frameword based on MVC design pattern

#### Features are 
  #### 1. Authintincation by using email, password and comformation email with verification url.
  #### 2. Reset password, with email verification mail.
  #### 3. Change user profile picture.
  #### 4. Add blog with Location, Description, one or more images and videos.
  #### 5. Add log and user can also comment on the blog just like facebook.
  #### 6. User can rate blog's user and also blog.
  #### 7. User can delete only his blog. -> Feature is implemented by the foreign key association of user_table and vlog_table.
  #### 8. Search blog in user personal section and can see all user blog with pagination and can also query with paggination.
